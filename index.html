Meshwork is alpha software and is not currently being maintained. Many features are missing, and many do not work properly. If you're interested in the project and would like to help, please contact me.

Meshwork creates a distributed peer-to-peer network that enables you to securely communicate with your friends. It supports chatting, file sharing, and more with a solid user interface that is both easy and fun to use.

Distributed, Peer-to-peer (P2P)
With Meshwork, you and all of your friends interconnect with each other to form a "mesh", so there's no central server that can go down.
Completely Secure
Your identity is only revealed to your trusted group of friends, so security and privacy are never a concern.
Cross Platform
Linux? OSX? Windows? No Problem. Meshwork runs perfectly and looks great on your platform of choice.
BitTorrent file transfers
Share files at blazing speeds without a second thought.
Firewall/Router Traversal
Meshwork does everything possible to work around your firewall or NAT router, so you and your friends will always be able to connect.
Open Source
Meshwork is open source, which means that anyone is free to download the source code, learn how it works, and make changes. Most importantly, since the code isn't hidden from the world, you can feel confident that Meshwork provides true security, and not just security through the obscurity of people not knowing how it works.
